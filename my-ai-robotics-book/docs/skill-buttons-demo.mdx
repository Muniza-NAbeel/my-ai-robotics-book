## Integration Instructions

To add these buttons to your own pages, import them like this:

```jsx
import {
  GlossaryButton,
  DiagramButton,
  TranslateButton,
  ExercisesButton,
} from '@site/src/components/SkillButtons';

// Then use in your MDX content:
<GlossaryButton term="Robot" />
<DiagramButton topic="Neural Network" />
<TranslateButton text="Hello world" />
<ExercisesButton chapter="Your chapter content here" />
```

Each button accepts a single prop:
- `GlossaryButton`: `term` - the technical term to define
- `DiagramButton`: `topic` - the concept to diagram
- `TranslateButton`: `text` - English text to translate to Urdu
- `ExercisesButton`: `chapter` - chapter content to generate questions from
