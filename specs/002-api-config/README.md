# API Configuration and Server Setup

This feature covers the essential setup instructions for the RAG chatbot application, including API key configuration, local server setup, and frontend integration.

## Overview

This feature provides step-by-step instructions for:
1. Setting up API keys for Cohere, Qdrant, and OpenAI
2. Running the FastAPI server locally
3. Integrating the Docusaurus frontend with the backend API

## Key Components

- Environment configuration
- Local development setup
- Frontend-backend integration
